# AI Resume Score Matcher

An AI-powered web application that analyzes resumes and matches them against job descriptions using natural language processing. The system generates a relevance score, highlights keyword matches and gaps, and provides recommendations to improve the resume ‚Äî helping job seekers instantly evaluate how well their resume fits a specific job posting.

## üöÄ Features

- **AI-Based Resume Matching**  
  Uses NLP/ML models to compare resumes with job descriptions.

- **Relevance Scoring System**  
  Generates a score (0‚Äì100%) based on skill match, experience relevance, and keyword alignment.

- **Keyword Extraction & Gap Analysis**  
  Identifies relevant skills and missing keywords.

- **Improvement Recommendations**  
  Personalized suggestions to increase match score.

- **Supports File Upload (PDF/DOCX) & Text Input**  
  Extracts text automatically.

- **Full-Stack Architecture**  
  Clean API separation with dedicated AI logic.
